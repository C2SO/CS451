main
    read file

read file  
    store file
    run process

run process
    start timer
    Every second, run check process

check process
    is there a current process running
        is it done?
            terminate current process
            start next process in wait list
    Is there a new process that has a higher priority?
        pause current process
        add it to a wait queue
        start new process
    are there more than one process that are added at this time?
        add other process with lower priority to the wait list
        pause current process
        start new process
    are there any process that will be added in the future?
        yes = wait
        no = exit program
